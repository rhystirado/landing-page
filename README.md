# Landing Page

## Description and Pre-Project Expectations

This is a mock landing page that can be modified to be used by a company or person. The aim of building this website is to become confident with using HTML and CSS to create a design that mimics an image of a website.

The website is inspired by an Odin Project assignment, and uses only HTML and CSS. It will initially be based on the image provided, however, it will be adapted by myself over time.

By completing this project I will demonstrate my knowledge of HTML and CSS. Flexbox is an important skill that I have learnt that will be used extensively for layouts in this website.
